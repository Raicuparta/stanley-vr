## How to Install

[Get StanleyVR on itch.io](https://raicuparta.itch.io/stanley-vr). You can also use the [**itch.io app**](https://itch.io/app) to install the mod. This way you can get automatic updates.

[![Search for StanleyVR on the itch.io app, and press the install button](https://user-images.githubusercontent.com/3955124/230877152-ab38b5a6-9449-4859-b87c-a8ac3a79537b.png)](https://raicuparta.itch.io/stanley-vr)

Run `RaiManager.exe` (or click Open in the itch.io app) and follow the instructions in the installer.

## Requirements

- A legitimate and up to date version of The Stanley Parable: Ultra Deluxe ([get it on Steam](https://store.steampowered.com/app/1703340/The_Stanley_Parable_Ultra_Deluxe/)). Note: the old, non-deluxe version isn't compatible with this mod.
- A PC ready for PCVR.
- An OpenVR-compatible VR headset. Examples:
  - Quest/Quest 2 connected to a PC via Link Cable, Air Link, Virtual Desktop, ALVR, etc
  - Rift/Rift S
  - Valve Index
  - Any VIVE
  - Any Windows Mixed Reality device (probably?)
- VR controllers. This isn't playable with a normal game controller, motion controls are required.

## How to Uninstall

Start the installer again (by starting RaiManager.exe or clicking Open in the itch.io app), and click the uninstall button.
